Suite de mejoras para WordPress para faciltar el trabajo a los desarrolladores web.

== Description ==

Un montón de funcionalidades para que hacer una página web sea cada vez más sencillp.

== Installation ==

1. Subir la carpeta del plugina al directorio `/wp-content/plugins/`
2. Activar el plugin desde la sección de plugins de WordPress
3. Activar las funcionalidades deseadas desde el panel de control del plugin
